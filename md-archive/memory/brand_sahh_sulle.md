## Brand baseline

- **Nimi:** Sähh sulle
- **Tegevus:** Toidukullerus rataste peal — jalg, ratas, scooter, mitte autod
- **Asukoht:** Tartu (tudengilinn)
- **Sihtgrupp:** Tudengid + noored, kes ei taha suurkorporatsioonide rakendusi (Bolt Food, Wolt)
- **Faas:** Uus startup, väike meeskond, eelarve madal
- **Brand-positioon:** anti-corporate, peer-to-peer, "kaaslane toob söögi"

## Brand-voice

**Tonaalsus:** "tõsiseltvõetav aga noor". Direct, friendly, no corporate fluff. Hääl = nagu kaaslane räägiks, mitte nagu ettevõte.

**Brand-pun:** "Sähh sulle" ise on argi-eestikeelne väljend ("here you go") — bränd ON keelelis-mängust sündinud. Mängu jätkamine on lubatud:
- "Hetkel ei sähhi" (404 lehel)
- "Sähhime kohale 30 minutiga"
- "Sähhid ise kaasa võtad?" (pickup-option)

## Visuaalne identiteet (kujunes 404 lehe loomisel)

### Värvilahendus

| Token | Hex | Roll |
|---|---|---|
| `--paper` | `#F0E8D6` | Takeaway-paper kreem — taust |
| `--paper-dim` | `#E5DCC4` | Sügavam paper-toon, sekundaarsetele aladele |
| `--ink` | `#1A1815` | Deep blackish — tekst, struktuur |
| `--mustard` | `#D9A437` | Brand signature — soe goldenrod, mitte Wolt-sinine ega Uber-roheline |
| `--cherry` | `#B7372D` | Aktsent — ketchup-toon, hoiatuse ja stamp'ide jaoks |
| `--olive` | `#5F633D` | Sekundaarne neutraalne — herb / rohelise lisand |

### Tüpograafia

- **Display:** Khand (Google Fonts, condensed strong sans) — bold-confident kaupluse-silt energy
- **Body:** Geist (Google Fonts, modern clean sans) — neutraalne tugi

### Visuaalsed signatuurid

- **Paper-bag** (takeaway kott) tilted -8° + offset shadow — brand asset
- **Circular VOID stamp** cherry-värvi, rotated, "404" või muu sõnumiga — uniseks ka muudes kontekstides
- **Sticker-style nupud** — offset mustard/cherry shadow taga, slightly rotated
- **Hand-drawn wavy underlines** ghost-button'idel — SVG paths

### Layout-printsiip

Golden ratio split (61.8 / 38.2). Visual area on suurem (paper-bag), text composition on väiksem. Ankurdused diagonaalsetes nurkades — brand-bar üleval-vasakul, kontakt all-paremal.

## Edaspidiseks

Indrek nägi neis tegelastes potentsiaali. Võimalikud tulevased tellimused:
- Landing page (Auto24-stiilis tellimissait, mitte rakendus)
- Brändi-täiustused (logo, värvi-kaart, social-templates)
- Kampaania-graafika (kevadised üritused, tudengi-aktsioonid)
- Print materjalid (menüüd, takeaway-stickerid päriselt)

**Säilita konsistentsus:** kui klient tagasi tuleb, kasuta sama värvilahendust + sama fonti + sama signature visual elements (paper-bag, sticker-buttons, hand-drawn underlines). Bränd peab kasvama, mitte iga projektiga uuesti sündima.
