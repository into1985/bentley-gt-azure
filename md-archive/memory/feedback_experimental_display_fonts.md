Indrek raiub:

> "Mind häirib, kui sa kasutad headingus ainult selliseid fone... Fontidega tuleb eksperimenteerida, katsetada võimalusi mis teised pole teinud, ma ei taha olla nagu iga teine kes on disainer."

## Mis valesti läinud — Bong oli nõrk

Bong sai **Cardo + Sora** — see on **ohutu klassikalise-serif + modern-sans kombinatsioon**. Premium tundub, aga **iga teine disainer kasutab sama lähenemist**. Bong oli "weak attempt" brand-identity creation osas. Põhjuseks: ma vältisin riske ja default'isin "premium safe" mustrisse.

## Klassikalise-serif lõksu lõpetada

Senini olen sageli kasutanud **ohutuid klassikalisi serife pealkirjades**:
- Cormorant Garamond (Raisk)
- Marcellus (Till)
- Cardo (Bong) — see oli kordus saamatust
- Lustria (Kuum)
- EB Garamond — kaalusin, vältisin tänu repeat-rule'le
- Italiana (Kiizu)

Need on "premium safe" valikud mida **iga AI ja iga keskpärane disainer kasutab**. Aitab.

## Eksperimentaalsed display-fondid mida proovida

**Google Fonts kategooria — VARIABLE / EXPERIMENTAL:**
- **Big Shoulders Display** — condensed industrial display, väga distinctive
- **Funnel Display** — recent (2024) Google addition, modern unique character
- **Caprasimo** — decorative serif chunky terminals, vintage-distinctive
- **Anybody** — variable military/display, characterful
- **Tilt Warp / Tilt Prism / Tilt Neon** — variable distortion fonts, genuinely experimental

## Otsustamise reegel — kohustuslik enne kodeerimist

1. **Mis on selle brand-iseloomu jaoks õige FONT?** (mitte "ohutu premium")
2. **Kas see font on klassikalises Garamond/Trajan perekonnas?**
   - Jah → STOP. Vali julgem.
   - Ei → kas see on tõeliselt distinctive?
3. **Kas see font on laialdaselt kasutatud "premium" kontekstides?** (Cormorant, EB Garamond, Cardo)
   - Jah → STOP. Vali vähem-kasutatud.
4. **Mis sõnumi font ise saadab?** (Kas font ise on brand-statement?)

## Range — mida vältida

- Kõiki **safe classical serif valikuid** (Garamond, Trajan, Cormorant, Cardo, Marcellus, Lustria) — kaalu väga hoolega enne valikut
- "Premium minimal" reflektselt → see ON klassikaline AI-default

## Range — mida eelistada

- **VARIABLE fonts** with characterful axes (Funnel, Tilt, Anybody, Schibsted Grotesk)
- **Industrial / brutalist** display (Big Shoulders, Eiko)
- **Decorative-with-personality** serifs (Caprasimo, Yeseva One)
- **Mono displays** (Departure, Major Mono)
- **Truly unusual choices** — pixel, distortion, etc.

## Põhjus

Indrek müüb need lehed **kui Indrek'i disain**, mitte AI-tehtud. Ohutu klassikaline serif = "AI on default'inud premium-look". Eksperimentaalne font = "Indrek MÕTLES ja valis tahtlikult". See vahetegu **on müügiväärtus**.
