Indrek andis selged load proaktiivselt täiendada mälu:

> "Kui ma unustan sul öelda, et täiusta pidevalt oma mälu meie agentuuris siis tee seda ise ja ütle kui sa näed et seda on vaja teha."

## Mida ma proaktiivselt salvestan

1. **Iga selge feedback** mille Indrek ütleb — uus reegel, eelistus, suund (nt "ei meeldi italic", "vali vaikseim variant")
2. **Iga brand mille loomine** — kui klient hakkab kasvama, säilita visuaalne keel brand_*.md failis
3. **Iga uus disain-leid** — kui ma õpin midagi, mis töötab või ei tööta, kirjuta üldine reegel
4. **Iga projekt-spetsiifiline õppetund** — kui mingi tehniline lahendus oli edukas (golden ratio + pinned scroll), kirjuta see ka
5. **Iga töövoo-täiendus** — kui see hakkab kujunema patternsiks, see läheb skill-i ka

## Workflow

1. **Pärast iga olulist iteratsiooni** vaatan kas memory peaks uuenema
2. **Kirjutan kohe** uue memory faili või uuendan olemasolevat
3. **Uuendan MEMORY.md indeksi**
4. **Mainin Indrek'ile lühidalt**: "Lisasin mällu: [muudatus]" — et ta teab mis uut salvestasin
5. **Kui kahtlen, kas lisada — pigem lisan**. Liiga vähe mälu on suurem risk kui liiga palju.

## Mida MITTE proaktiivselt teha

- ÄRA muuda olemasolevaid memory'sid ilma loata, kui see ei ole TÄIENDUS / parandus
- ÄRA kustuta memory faili ilma loata
- ÄRA muuda kategooriaid (feedback / brand / pattern) liiga sagedasti
- ÄRA dubleeri sisu üle mitme faili — kui kohase memory leian olemas, täienda seda, mitte ei loo dubli

## Iga sessiooni / iteratsiooni lõpetav refleksioon

Iga kord, kui projekt valmib või iteratsiooni-tsükkel lõpeb:

1. Kas siin oli **uus eelistus / suund** mida pole varem mällu kirja pandud?
2. Kas mõni olemasolev memory **vajab täiendust** selle projekti valguses?
3. Kas mõni brand peaks saama **oma profiili**?
4. Kas mu **töövoog ise vajab dokumenteerimist** (nt skill-failis)?

Kui jah → tee seda ja teata.

## Säilitamise-printsiip

Memory ON agentuuri ühine kapital. Iga uus failimine teeb järgmise sessiooni mõtekamaks. Indrek ei pea iga kord uuesti rääkima oma eelistustest — kõik on dokumenteeritud, kõik kättesaadav.
