Indrek märkis:

> "Alati ei pea h1 alla panema mingit 'luulelist' juttu, kohati on see labene tulnud, võib ju lihtsalt viisakalt ka öelda, et lehte ei leitud vms."

## Mida tähendab praktikas

Senini olen iga 404 lehe juures otsinud **kontseptuaalse metafoori** brand-nimest:

| Brand | Metafoor | Hinnang |
|---|---|---|
| Välk | "Välk lõi mööda" | ✓ geniaalne (nimi = sõna) |
| Pauk | "Märklaud kadunud" | ✓ tugev |
| Kurk | "Kurk veeres ära" | ✓ geniaalne (kasutaja lemmik) |
| Tigu | "Pakk ei jõudnud kohale" | ✓ tugev |
| Inzain | "Kaader puudu" | ✓ on-brand |
| Tudi | "Leht magab" | ✓ on-brand |
| Minna | "Pole kuhu minna" | ✓ geniaalne |
| Sähh sulle | "Hetkel ei sähhi" | ✓ on-brand |
| Raisk | "See aadress ei vastanud" | ✓ — see on tegelikult VIISAKAS, mitte metafoor! |
| Kiizu | "See on privaatne" | ✓ neutraal, polise |
| Kuus | "Pole ehitatud" | ✓ agentuuri jaoks puhtalt sobiv |
| Musi | "Maasikad magavad" | ~ borderline labane / cheesy |

Mõned brandid (Kurk, Tigu, Minna) on **päris-sõnad** ja nende metafoor tuli loomulikult. Need on geniaalsed.

Mõned brandid on **välja-mõeldud nimed** (Välk, Pauk, Raisk, Kuus) ja seal metafoor on **forced**. Mõnel õnnestus (Pauk — "märklaud kadunud" sobib täiesti), mõnel hakkab tunduma "AI on otsinud sõnamängu".

Musi puhul "Maasikad magavad" on **borderline labane** — sundtud poeetilisus mis pole hädavajalik.

## Reeglid alates nüüd

### Headline-strateegia otsuse protokoll

**1. Kas brand-nimi on PÄRIS eesti sõna?**
- Jah → proovi sõnamängu, AGA vali ainult kui see on **tõeliselt elegantne** (mitte sundtud)
- Ei → kasuta **viisakat, otsest sõnumit**: "Lehte ei leitud", "Pole sellist lehte", "404 — aadress puudub"

**2. Kas brand-tonaalsus nõuab metafoori?**
- Mänguline / loov brand → metafoor sobib
- Konservatiivne / formaalne / luxury / wellness → **otsene sõnum on parem** (vt Raisk näide — formaalne ja viisakas, mitte metaforiline)

**3. Kas metafoor ütleb midagi rohkemat kui "lehte ei leitud"?**
- Jah → lisa väärtust, hoia
- Ei → drop it, lihtsam on tugevam

**4. Kui metafoor tundub "labane" / cheesy** — KOHE drop ja vali otsene polise sõnum.

## Näited heast viisakast copy'st

- "Lehte ei leitud."
- "Pole sellist lehte."
- "404 — aadress puudub."
- "Ei leidnud teie otsitut."
- "Otsige uuesti."
- "See lehekülg ei eksisteeri."

Need ON **TUGEVAD**. Pole vaja luulekat ümberkääpamist iga kord.

## Disaini-mõju

Kui headline on viisakas/lihtne, **disain peab kandma loovuse** (palett, fondid, illustratsioon, motion). Headline ei pea olema atrakitne — atrakitsioon tuleb tervikust.

Kui headline on metafoorne/luuleline, see **võtab fookuse** ja disain saab olla rahulikum.

Tasakaal: ei pea olema kõik elemendid "kõva". Vahel **headline on rahulik, disain on dramaatiline** — see töötab.
